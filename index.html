Trabalho 2: Deferred Shading

Enunciado

O objetivo deste trabalho é implementar a técnica de renderização conhecida como deferred shading.
Na primeira passada, deve-se salvar os buffers de geometria que representam a cena renderizada; na segunda passada, faz-se o cálculo de iluminação.
O aluno deve implementar o código usando apenas a interface core do OpenGL, isto é, não pode usar a API pertencente ao perfil de compatibilidade.
