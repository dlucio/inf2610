INF2610 - Rendenring em Tempo Real

Trabalho 1: Mapa de Rugosidade

Enunciado

O objetivo deste trabalho é implementar a técnica de renderização conhecida como mapa de rugosidade (bump mapping).
O aluno deve implementar o código usando apenas a interface core do OpenGL, isto é, não pode usar a API pertencente ao perfil de compatibilidade.
